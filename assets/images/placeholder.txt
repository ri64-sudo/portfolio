## Image Guidelines:
- Use high-quality images with good resolution
- Ensure images are optimized for web (compressed but clear)
- Use consistent aspect ratios for project carousel images
- Include descriptive alt text for accessibility
- Consider using placeholder images from services like Unsplash or Pexels

## Placeholder Image Services:
- https://picsum.photos/ (Random placeholder images)
- https://via.placeholder.com/ (Custom placeholder images)
- https://unsplash.com/ (Free stock photos)
